# Vim Key Notation Reference

## Modifiers

| Notation | Key     | Example              |
|----------|---------|----------------------|
| `<C-x>`  | Ctrl+x  | `<C-s>` = Ctrl+s     |
| `<A-x>`  | Alt+x   | `<A-j>` = Alt+j      |
| `<M-x>`  | Meta+x  | Same as Alt          |
| `<S-x>`  | Shift+x | `<S-l>` = Shift+l    |

## Special Keys

| Notation        | Key                                     |
|-----------------|-----------------------------------------|
| `<cr>`          | Enter (Carriage Return)                 |
| `<esc>`         | Escape                                  |
| `<tab>`         | Tab                                     |
| `<bs>`          | Backspace                               |
| `<del>`         | Delete                                  |
| `<space>`       | Spacebar                                |
| `<leader>`      | User-defined (usually `\` or `<space>`) |
| `<localleader>` | Buffer-local leader                     |

## Navigation Keys

| Notation     | Key         |
|--------------|-------------|
| `<up>`       | Arrow up    |
| `<down>`     | Arrow down  |
| `<left>`     | Arrow left  |
| `<right>`    | Arrow right |
| `<home>`     | Home        |
| `<end>`      | End         |
| `<pageup>`   | Page Up     |
| `<pagedown>` | Page Down   |

## Function Keys

| Notation         | Key           |
|------------------|---------------|
| `<F1>` - `<F12>` | Function keys |
| `<S-F1>`         | Shift+F1      |

## Combined Modifiers

| Notation  | Key          |
|-----------|--------------|
| `<C-S-a>` | Ctrl+Shift+a |
| `<C-A-j>` | Ctrl+Alt+j   |

## Mouse (GUI/Terminal with mouse support)

| Notation            | Action            |
|---------------------|-------------------|
| `<LeftMouse>`       | Left click        |
| `<RightMouse>`      | Right click       |
| `<2-LeftMouse>`     | Double left click |
| `<ScrollWheelUp>`   | Scroll up         |
| `<ScrollWheelDown>` | Scroll down       |
