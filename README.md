# To People Who Clone This Repo

_brace yourself._

There is no god here.

There is only hope.

Hope that it boots.

Hope that the plugins load.

Hope that the LSP attaches.

Hope that whatever just exploded will somehow unexplode if you restart Neovim hard enough.

This config works on my machine in the same way a shopping cart with one bad wheel technically still moves.

Sometimes it flies.

Sometimes it screams.

Sometimes it opens twelve windows and teaches me new forms of regret.

I am not willing to fix all of it because suffering builds character and this repo is, above all else, a character-building exercise.

Nothing will save you here except your own sanity.

Not Mason.

Not Lazy.

Not JDTLS.

Not treesitter.

Not prayer.

If it works, do not question it.

If it breaks, act natural.

If it completely disintegrates in front of you, that is not a bug.

That is the full intended experience.

Embrace the suck.

Become the suck.

Transcend through the suck.
